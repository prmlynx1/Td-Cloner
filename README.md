# ⚡ TrollDeath Server Cloner

A powerful **Discord Server Cloning Tool** that allows you to quickly copy the structure of a server (roles, categories, and channels) into another server.

The tool also supports **server name and icon cloning** to recreate the original server layout automatically.

## 🚀 How To Use

1️⃣ Run **Cloner.exe**

2️⃣ Enter the required information:

* Discord Token
* Source Server ID (server you want to copy)
* Target Server ID (your server)

3️⃣ Choose the option:

```
[1] Full Auto Clone
```

The tool will automatically:

* Delete existing roles and channels
* Copy roles from the source server
* Copy categories and channels
* Clone server name and icon
